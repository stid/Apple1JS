## Shape

Browser-based, cycle-accurate Apple 1 emulator (TypeScript + React, built with Vite). The 6502 CPU
is a **dual engine**: a TypeScript reference implementation and a Rust→WASM implementation
(`wasm-cpu/` compiled via `wasm-pack` into `src/wasm/`), switchable at runtime with the JS engine
as the always-available fallback/reference oracle. Layered architecture: a framework-free emulation
core, a worker-hosted system-integration layer, and a React UI that consumes inspectable state.

## Entry points

- **Dev:** `yarn dev` → `wasm:build:release` then `vite` → boots `index.html` → `src/index-web.tsx`
- **Build:** `yarn build` → `wasm:build:release` + `tsc` + `vite build`
- **WASM:** `yarn wasm:build:release` (`cd wasm-cpu && wasm-pack build --release --target web --out-dir ../src/wasm`)
- **Tests / gate:** `yarn test` (all) · `yarn test:ci` (lint + type-check + `vitest run`)

## Surfaces in use

`RENDER` (the CRT/terminal display + debugger UI) and `none` (internal emulation logic / invariants).
No `CLI` / `HTTP` / `MCP` / `DB` surfaces. The headless `BENCH=1` throughput harness is a measurement
tool, not a product surface.

## Invariants (always-true rules for THIS project)

> Recurring rules that hold across all work here (LCD principle: don't relitigate them each
> time). The Deep lane's Constitution check and the red-green loop honor these.

- **Never commit to `master`** — always a feature branch (`feat/ fix/ refactor/ docs/ test/ chore/`).
- **Dual-engine parity** — JS and WASM engines must stay feature-equal; test **both** for every
  engine change. The JS engine is the reference oracle / fallback.
- **No `any`** (strict mode), **no `console.log`** (use `LoggingService`), **no hardcoded colors**
  (design tokens — except the CRT display).
- **Bump `src/version.ts`** before every PR (single source of truth; patch/minor/major per branch prefix).
- **Stateful/inspectable contracts** (`IVersionedStatefulComponent`, `IInspectableComponent`) are
  scoped to the **core/IO layer**, not React presentational components.
- **WASM build is speed-first** (`opt-level = 3`, ~155 KB); don't restore the abandoned size-first
  (<100 KB) target without approval.
